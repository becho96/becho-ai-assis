# Quick Start: Telegram Dialogue Analysis

## 🚀 Быстрая настройка (5 минут)

### Шаг 1: Авторизация в Telegram

```bash
npx tsx scripts/auth-telegram-user.ts
```

Следуйте инструкциям:
1. Введите номер телефона (с кодом страны, например +79123456789)
2. Введите код из SMS
3. Скопируйте session string

### Шаг 2: Сохраните session string

```bash
echo "TELEGRAM_SESSION_STRING=<скопированный_session_string>" >> .env
```

⚠️ **Важно:** Session string = полный доступ к вашему Telegram аккаунту. Храните в `.env`, не коммитьте в git!

### Шаг 3: Тест

```bash
npx tsx scripts/test-dialogue-analyzer.ts @username 7
```

Замените `@username` на реальный username из ваших контактов.

## 💬 Использование через Telegram бота

Теперь вы можете отправлять боту запросы:

### Примеры

**Полный анализ:**
```
Проанализируй переписку с @zadum_off за последние 7 дней
```

**Конкретный период:**
```
Что обсуждали с @colleague на этой неделе?
```

**Извлечение задач:**
```
Сформируй список задач из переписки с клиентом
```

## 🔧 Troubleshooting

### "User not found"
- Проверьте написание username
- Убедитесь что есть переписка с этим пользователем

### "No messages found"
- Попробуйте больший период: "за последние 30 дней"

### "Rate limit"
- Подождите 5 минут и попробуйте снова
